Mis Proyectos

Bot de Trading Automatizado

Python — PostgreSQL — MetaTrader 5

Bot que ejecuta operaciones de compra y venta de forma autónoma en el mercado de divisas (Forex). Se conecta a MetaTrader 5 para obtener datos de precio en tiempo real, aplica estrategias basadas en indicadores técnicos y registra cada operación en PostgreSQL.

Análisis de Apuestas Deportivas

Python — PostgreSQL — Tkinter

Aplicación de escritorio que recopila y analiza datos de resultados deportivos para identificar patrones estadísticos. Interfaz gráfica en Tkinter con filtros, métricas y almacenamiento del historial en PostgreSQL.

Sistema de Facturación

C# — SQL Server — Windows Forms

Sistema de gestión empresarial para emisión y control de facturas. Permite registrar clientes, productos e impuestos, generar comprobantes y mantener historial completo de transacciones en SQL Server.

Plataforma de Préstamos

PHP — MySQL

Plataforma web para gestión de solicitudes y seguimiento de préstamos. Incluye autenticación de usuarios, cálculo de cuotas e intereses, panel administrativo y reportes de cartera almacenados en MySQL.
